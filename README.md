# Small Web Projects

A collection of mini frontend web development projects made using HTML, CSS, and JavaScript. These projects help practice core concepts in UI design, interactivity, layout, and animation.

---

## Projects

### 🕒 Digital Clock
A real-time digital clock that displays hours, minutes, and seconds, updating every second.

- **Tech Used:** HTML, CSS, JavaScript
- **Features:** Live time updates with styled digital format

---

### 🐭 Cursor Tail Animation
A creative animation that adds a colorful trailing effect behind the cursor, creating a dynamic and playful UI experience.

- **Tech Used:** HTML, CSS, JavaScript
- **Features:** Mouse tracking, animation effects

---

### ✅ Form Input Validation
A basic HTML form with real-time validation using JavaScript to check input fields like name, email, and password.

- **Tech Used:** HTML, CSS, JavaScript
- **Features:** Real-time validation, error highlighting, UX feedback

---

### 🖼️ Master Page Layout
A responsive and reusable web page layout with header, footer, and main sections, suitable for use as a basic template or landing page.

- **Tech Used:** HTML, CSS
- **Features:** Responsive design, semantic structure, clean layout
